Comparative Analysis of the Beef Sector in Ireland
This repository contains the code and data for a rigorous comparative analysis of the beef sector in Ireland. The project aims to identify key factors that have influenced the Irish beef market in the past and present, and develop forecasting models to make informed projections about future beef prices.

Contents
data/: This folder contains the raw data used in the analysis, including beef trade prices, land usage, sustainability practices, population and employment trends, climate indicators, and investment trends. cleaned_data/: This folder contains cleaned and processed data used in the analysis. code/: This folder contains the code used to perform the analysis, including data cleaning, processing, and modeling. results/: This folder contains the results of the analysis, including visualizations and forecasting models. README.md: This file provides an overview of the project and instructions for using the repository.

Requirements
To run the code in this repository, you will need to have the following software and packages installed:

Python 3.6 or higher Jupyter Notebook Pandas Matplotlib Scikit-learn

Usage
To use this repository, follow these steps:

Clone the repository to your local machine. Install the required software and packages listed above. Open the Jupyter Notebook files in the code/ folder and run the cells to reproduce the analysis and results. View the visualizations and forecasting models in the results/ folder.
